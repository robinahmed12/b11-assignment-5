Question-2: What is the difference between innerHTML, innerText, and textContent ?

differences:- 1. innerHTML Gets or sets HTML content, including tags. Parses HTML if used to set
2. innerText Gets or sets only visible text inside the element. Ignores hidden elements and respects CSS styles like display: none;.
3. textContent Gets or sets all text content, including hidden elements. Ignores HTML tags and returns raw text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element to handle events on its child elements. This works because of event bubbling, where events propagate from the target element up to the root.
